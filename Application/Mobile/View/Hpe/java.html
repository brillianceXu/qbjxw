  • 编程性语言 适用于手机端和电脑端 与之相关专业：计算机科学与技术、软件工程、电子商务、计算机信息管理、电子信息工程、光电信息科学工程、电子科学与技术、通信工程、机械及自动化相关专业。

    专业前景

    Java是目前世界上最流行的计算机编程语言，Java软件工程师的就业机会和前景是非常大的。再加上Java软件工程师不仅IT专业企业需要,广大的非IT企业也需要。前景是非常乐观的。

    课程体系

    在Java课程培训周期内，培养学员具备企业级软件项目分析与研发经验，能够合理运用Java语言与语法，掌握面向对象的开发与设计思想；理解Java主流开发技术框架，深入了解 B/S程序的特点，能够熟练使用JavaEE平台技术，开发出高性能高复用性的JavaEE企业级应用程序。通过学习任务完成情况及考试成绩，在综合评定后将会获得课程认证证书。

    从事岗位及薪资待遇

    可从事岗位：Java研发工程师，Java游戏开发工程师，Java后端工程师，大数据分析师，Java运维工程师，及JavaEE工程师等。

    根据IDC的统计数字，在所有软件开发类人才的需求中，对Java工程师的需求达到全部需求量的60%—70%。同时，Java工程师的薪水相对较高。Java软件工程师一般月薪范围在4k—10k，远远超过了应届毕业生月薪2.5k的平均水平。通常来说，有一年工作经验的Java软件工程师的薪酬大致在月薪8k—10k左右。

    职业发展：

    一是成为管理人员，例如产品研发经理，技术经理，项目经理等；二是延续技术工作之路，成为高级软件工程师、需求工程师，软件架构师等。

  • 软件测试（Software Testing），描述一种用来促进鉴定软件的正确性、完整性、安全性和质量的过程。换句话说，软件测试是一种实际输出与预期输出间的审核或者比较过程。软件测试的经典定义是：在规定的条件下对程序进行操作，以发现程序错误，衡量软件质量，并对其是否能满足设计要求进行评估的过程。

    软件测试技术是软件开发过程中的一个重要组成部分，是贯穿整个软件开发生命周期、对软件产品（包括阶段性产品）进行验证和确认的活动过程，其目的是尽快尽早地发现在软件产品中所存在的各种问题——与用户需求、预先定义的不一致性。检查软件产品的bug。写成测试报告，交于开发人员修改。

    专业前景

    近几年来，中国软件业保持了每年超过30%的高速发展态势，但整体规模与软件业发达的国家间仍有着较大差距，在软件业发达国家，软件测试人员与开发人员之比接近1:1，而在中国该比例仅为1:8左右，很多企业没有专门的测试职位。业内专家预计，在未来5到10年中，我国IT业对软件测试人才的需求数字还将继续增大。随着测试重要性的日趋突显，我国软件测试人才正处于一个“双高”地位，即地位高、待遇高，职场前景非常广阔。

    课程体系

    通过软件测试课程培训，培养学员具备软件测试的实际项目经验，熟练掌握软件测试必须的基础理论知识及测试技能，理解测试设计思想及原理，熟练使用软件测试工具完成项目的测试任务。能够运用所学的测试基础知识分析需求与设计测试用例，编写测试缺陷报告及项目报告；掌握测试工具（ALM、UFT、LoadRunner等）的实际应用，具备手工测试、功能自动化测试及性能测试项目的实践经验，能够达到一年以上的测试工程师所具备的测试技术水平。通过学习任务完成情况及考试成绩，在综合评定后将会获得课程认证证书。

    开发测试工程师，功能测试工程师，自动化测试工程师，性能测试工程师，安全测试工程师等。目前国内软件测试人才缺口高达30万，已成为我国软件产业发展的瓶颈之一。在大型软件开发企业中，软件测试工程师基本处于“双高”地位，即地位高、待遇高，月薪可高达7k—8k。中小规模软件企业月薪在4k—6k左右的居多。

    职业发展：

    软件测试人才更强调经验积累，在几年的测试经验背景下，可以逐步转向管理或者资深测试工程师，担当测试经理或者部门主管。

  • 大数据（big data），指无法在一定时间范围内用常规软件工具进行捕捉、管理和处理的数据集合，是需要新处理模式才能具有更强的决策力、洞察发现力和流程优化能力的海量、高增长率和多样化的信息资产。

    大数据的出现简单的讲是海量数据同完美计算能力结合的结果。确切的说是移动互联网、物联网产生了海量的数据，大数据计算技术完美地解决了海量数据的收集、存储、计算、分析的问题。大数据时代开启人类社会利用数据价值的另一个时代。

    专业前景

    大数据是新兴技术领域，是信息技术与专业技术、信息技术产业与各行业领域紧密融合的典型领域，有着旺盛的应用需求、广阔的应用前景。未来是一个数据的时代，也是数据科学家的时代!

    课程体系

    通过编程语言与大数据知识理论学习、上机案例实践与企业级大数据项目实战，使学员快速掌握大数据的基础原理及实际应用；课程内容设计由浅入深，逐步提升学生学习效果；通过实训课程的学习与实践，掌握大数据常用关键技术与工具框架的配置应用，具备大数据分析与数据管理的能力。同时，培养学员养成独立解决问题的能力。通过学习任务完成情况及考试成绩，在综合评定后将会获得课程认证证书。
